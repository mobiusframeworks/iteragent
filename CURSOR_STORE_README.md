# 🚀 InterTools - Cursor Store Agent

## 📦 One-Click Installation

```bash
# Install from Cursor Store
curl -sSL https://raw.githubusercontent.com/luvs2spluj/intertools/main/cursor-store-install.sh | bash

# Or install directly
npm install -g intertools@latest
npx intertools@latest setup-cursor --all
```

## 🎯 What is InterTools?

InterTools is a **multi-agent orchestration system** designed specifically for Cursor IDE. It features 5 specialized agents working in coordination to dramatically speed up your development workflows.

### ✨ Key Features

- **🤖 5 Specialized Agents** working in perfect coordination
- **📊 Real-time Monitoring** of console and terminal logs
- **💬 Compact Cursor Chat Integration** with intelligent summaries
- **🔍 Intelligent Log Analysis** and error interpretation
- **⚡ 25-60% Faster Development** workflows
- **🎯 Agent Zero Integration** for enhanced AI capabilities
- **🔄 Continuous Monitoring** loops for immediate issue detection

---

## 👥 Specialized Agent Team

### 1. **Console Log Harvester**
- **Purpose**: Captures and analyzes console output in real-time
- **Frequency**: Every 5 seconds
- **Capabilities**: Console monitoring, log capture, error detection, pattern recognition

### 2. **Terminal Log Monitor**
- **Purpose**: Tracks terminal commands and output
- **Frequency**: Every 5 seconds
- **Capabilities**: Terminal monitoring, command tracking, output analysis, process monitoring

### 3. **Cursor Chat Communicator**
- **Purpose**: Sends compact summaries to Cursor AI chat
- **Frequency**: Every 15 seconds
- **Capabilities**: Chat integration, message formatting, priority routing, actionable suggestions

### 4. **Log Interpreter**
- **Purpose**: Analyzes logs and extracts meaningful insights
- **Frequency**: Every 10 seconds
- **Capabilities**: Log analysis, error interpretation, context understanding, pattern matching

### 5. **Code Change Suggester**
- **Purpose**: Provides actionable code suggestions for fixes
- **Frequency**: On-demand
- **Capabilities**: Code analysis, fix suggestions, implementation guidance, Anthropic integration

## 🚀 Quick Start

### 1. **Install InterTools**
```bash
npm install -g intertools@latest
```

### 2. **Setup Cursor Integration**
```bash
npx intertools@latest setup-cursor --all
```

### 3. **Start Multi-Agent System**
```bash
npx intertools@latest orchestrator --start
npx intertools@latest seamless --start
```

### 4. **Monitor Your Agents**
```bash
npx intertools@latest orchestrator --status
npx intertools@latest orchestrator --agents
npx intertools@latest orchestrator --logs
```

## 🔧 Commands Reference

### **Core Commands**
```bash
# Start orchestrator (Big Gun Mode)
npx intertools@latest orchestrator --start

# Enable Agent Zero integration
npx intertools@latest seamless --start

# Check status
npx intertools@latest orchestrator --status

# View specialized agents
npx intertools@latest orchestrator --agents

# Monitor logs
npx intertools@latest orchestrator --logs

# Stop everything
npx intertools@latest orchestrator --stop
```

### **Setup Commands**
```bash
# Setup Cursor integration
npx intertools@latest setup-cursor --all

# Check version
npx intertools@latest version

# Update InterTools
npx intertools@latest update
```

## 🔗 API Endpoints

When running, InterTools provides these APIs:

- **Logs API**: http://localhost:50005/api/logs
- **Cursor Chat API**: http://localhost:50005/api/cursor-chat
- **Agent Zero**: http://localhost:50001
