🚀 InterTools Web Chat Demo

📝 What is InterTools Web Chat?

InterTools Web Chat allows you to provide feedback about any web page by clicking on elements and sharing your thoughts. This feedback is then fed back to the InterTools system for analysis and improvement.

🎯 How to Use

  1. Start the Web Chat Server: Run npx intertools@1.0.13 web-chat --start
  2. Inject the Extension: Open browser developer tools (F12) and paste the extension script
  3. Click to Chat: Click the chat button that appears on the page
  4. Select Elements: Click on any element to highlight it for context
  5. Share Feedback: Type your thoughts about the page and send

💡 Features

  • Click-to-Chat: Click anywhere on the page to open the chat interface
  • Element Context: Select specific elements to provide targeted feedback
  • Page Analysis: Automatic capture of page context, viewport, and performance data
  • Real-time Communication: Instant feedback to InterTools system
  • Cross-Platform: Works on any website with the extension script
