🚀 InterTools Universal Click-to-Chat

Click any element on any website to capture HTML and send to Cursor

🎯 What This Does

This tool allows you to click on any element on any website and instantly capture its HTML code, add your feedback, and send it directly to Cursor AI for analysis and improvement suggestions.

✨ Features

🖱️ Universal Click

Works on any website - just click any element to capture it

📄 HTML Capture

Instantly captures the complete HTML code of clicked elements

💬 Smart Feedback

Add your thoughts and suggestions about the element

🤖 Cursor Integration

Formatted messages ready to paste into Cursor AI chat

📋 One-Click Copy

Copy HTML code or formatted messages to clipboard

🎨 Visual Highlighting

See exactly which element you're capturing with visual highlights

🚀 Quick Start

Step 1: Start InterTools Server

Make sure the InterTools web chat server is running:

npx intertools@1.0.14 web-chat --start

Step 2: Inject the Script

Step 3: Start Clicking!

Click on any element on the page. A chat box will appear with the HTML code and a message field. Add your feedback and send to Cursor!

⌨️ Keyboard Shortcuts

  • Escape - Close the chat box
  • Ctrl + Enter - Send message to Cursor
