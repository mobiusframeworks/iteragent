## 🚀 Installation Methods

### Method 1: Copy & Paste (Recommended)
```javascript
// 1. Copy contents of intertools-free-complete.js
// 2. Open any website
// 3. Press F12 → Console tab  
// 4. Paste and press Enter
// 5. Look for 🛠️ button!
```

### Method 2: Browser Extension
```bash
# 1. Download intertools-chrome-extension.zip
# 2. Extract files
# 3. Chrome → Extensions → Developer mode
# 4. Load unpacked → select folder
# 5. Works on all websites automatically!
```

### Method 3: Bookmarklet
```javascript
javascript:(function(){var s=document.createElement('script');s.src='https://your-domain.com/intertools-free.js';document.head.appendChild(s);})();
```
