# 🚀 Quick Install InterTools v1.0.13

## One-Line Installation

```bash
# Install InterTools in any environment
curl -sSL https://raw.githubusercontent.com/luvs2spluj/intertools/main/install-intertools.sh | bash
```

## Alternative Installation Methods

### Method 1: NPX (Recommended - No Installation Required)
```bash
npx intertools@1.0.13 setup-cursor --all
```

### Method 2: Global Install
```bash
npm install -g intertools@1.0.13
npx intertools@1.0.13 setup-cursor --all
```

### Method 3: Local Install
```bash
npm install intertools@1.0.13
npx intertools@1.0.13 setup-cursor --all
```

## 🎮 Interactive Cursor AI Commands

### What's New in v1.0.13
- ✅ **Working selection mechanism** in Cursor AI chat
- ✅ **Interactive error resolution** with command-based workflow
- ✅ **Smart error detection** and automatic command generation

### How to Use
1. **Start InterTools**: `npx intertools@1.0.13 orchestrator --start`
2. **Open Cursor AI chat** and look for `intertools-interactive.md`
3. **Select a command ID** from the available list
4. **Type in chat**: "Execute command: [COMMAND_ID]"
5. **Cursor AI will run the command** and provide detailed results

### Available Command Types

## 🔧 Troubleshooting

### Permission Issues
If you get permission errors, try:
```bash
# Use npx instead
npx intertools@1.0.12 --help

# Or install locally
npm install intertools@1.0.12
```

### Cursor Integration Issues
```bash
# Manual setup
npx intertools@1.0.12 setup-cursor --all

# Check if files were created
ls -la .cursor/inbox/
```

```bash

npx intertools@1.0.12 version
```
