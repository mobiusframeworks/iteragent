## 🚀 Installation Methods

### Method 1: NPM (Recommended)
```bash
npm install -g iteragent-cli
```

### Method 2: GitHub Clone
```bash
git clone https://github.com/luvs2spluj/iteragent.git
cd iteragent
./install.sh
```

### Method 3: Cursor Store Script
```bash
curl -sSL https://raw.githubusercontent.com/luvs2spluj/iteragent/main/cursor-install.sh | bash
```

## 🎮 Usage in Cursor

1. **Initialize in your project**:
   ```bash
   iteragent init
   ```

2. **Start the iterative loop**:
   ```bash
   iteragent run
   ```

3. **Press Enter** to send fix requests to Cursor's AI
